# OpenGL
These are assignments for school where I took starter code and added functionality to the code and 3D scene.

2-2_Assignment I transformed the pyramid so one pyramid is ontop of the other from being side by side.

3-2_Assignment I added simple shapes, applied scaling, rotating, and positional transformations, and applied a custom color for each shape to match a picture that was provided.

4-2_Assignment I added user input to traverse the 3D scene. I added q and e for up and down movement, scroll wheel to control the movement speed, and p and o to control the view (p for perspective view and o for orthographic).

5-2_Assignment I added custom textures to the simple shape meshes.

6-2_Assignment I added lighting to the 3D scene.
